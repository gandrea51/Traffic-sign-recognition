# 🛑 Traffic Sign Recognition

This project involves traffic sign classification. The work analyzes the transition from standard convolutional neural networks to more complex networks.

## 🗝️ Features

-  **Multi-Class Classification**: 43 types of road signs.
-  **Architectures**: 3 different types of CNN.
-  **Out-of-Distribution Testing**: Robustness testing on real images.
